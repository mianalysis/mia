MIA_IdentifyObjects

Description

Uses connected component labelling to convert a binary image to objects. Connectivity must be set to either 6 or 26.

Parameters

String inputImageName
  • String outputObjectsName
  • boolean whiteBackground
  • boolean singleObject
  • int connectivity
  • boolean showObjects
