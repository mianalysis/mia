Relate many-to-one

Description

Parameters

  • Parent (larger) objects (default = "")

  • Child (smaller) objects (default = "")

  • Method to relate objects (default = "Matching IDs")

  • Reference mode (default = "Centroid")

  • Child objects to test against (default = "")

  • Limit linking by distance (default = "false")

  • Maximum linking distance (px) (default = "1.0")

  • Inside/outside mode (default = "Inside and outside")

  • Minimum percentage overlap (default = "0.0") Percentage of total child volume overlapping with the parent object.

  • Require centroid overlap (default = "true")

  • Only link objects in same frame (default = "true")

  • Calculate fractional distance (default = "true")

  • Enable multithreading (default = "true")
