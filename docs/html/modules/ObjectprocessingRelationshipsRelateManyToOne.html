Relate many-to-one

Description

Parameters

  • Parent (larger) objects
    • Type: InputObjectsP

  • Child (smaller) objects
    • Type: InputObjectsP

  • Method to relate objects
    • Type: ChoiceP
    • Default value: Matching IDs
    • Choices:
      • Matching IDs
      • Proximity
      • Proximity to children
      • Spatial overlap

  • Reference point
    • Type: ChoiceP
    • Default value: Centroid
    • Choices:
      • Centroid
      • Surface
      • Child centroid to parent surface

  • Child objects to test against
    • Type: ChildObjectsP

  • Limit linking by distance
    • Type: BooleanP
    • Default value: false

  • Maximum linking distance (px)
    • Type: DoubleP
    • Default value: 1.0

  • Inside/outside mode
    • Type: ChoiceP
    • Default value: Inside and outside
    • Choices:
      • Inside and outside
      • Inside only (set outside to zero)
      • Outside only (set inside to zero)

  • Minimum percentage overlap
    • Description: Percentage of total child volume overlapping with the parent object.
    • Type: DoubleP
    • Default value: 0.0

  • Require centroid overlap
    • Type: BooleanP
    • Default value: true

  • Only link objects in same frame
    • Type: BooleanP
    • Default value: true

  • Calculate fractional distance
    • Type: BooleanP
    • Default value: true

  • Enable multithreading
    • Type: BooleanP
    • Default value: true
