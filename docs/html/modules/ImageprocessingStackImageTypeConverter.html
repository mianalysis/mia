Image type converter

Description

Parameters

  • Input image
    • Type: InputImageP

  • Apply to input image
    • Type: BooleanP
    • Default value: true

  • Output image
    • Type: OutputImageP

  • Output image type
    • Type: ChoiceP
    • Default value: 8-bit integer
    • Choices:
      • 8-bit integer
      • 16-bit integer
      • 32-bit float

  • Scaling mode
    • Type: ChoiceP
    • Default value: Clip (direct conversion)
    • Choices:
      • Clip (direct conversion)
      • Fill target range (normalise)
      • Scale proportionally
