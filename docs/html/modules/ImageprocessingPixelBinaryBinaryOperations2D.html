Binary operations 2D

Description

Expects black objects on a white background. Performs 2D fill holes, dilate and erode using ImageJ functions.

Parameters

  • Input image
    • Type: InputImageP
  • Apply to input image
    • Type: BooleanP
    • Default value: true
  • Output image
    • Type: OutputImageP
  • Filter mode
    • Type: ChoiceP
    • Default value: Dilate
    • Choices:
      • Dilate
      • Distance map
      • Erode
      • Fill holes
      • Outline
      • Skeletonise
      • Watershed
  • Number of iterations
    • Type: IntegerP
    • Default value: 1
