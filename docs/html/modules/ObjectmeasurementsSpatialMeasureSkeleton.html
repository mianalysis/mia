Measure skeleton

Description

Uses the AnalyzeSkeleton plugin by Ignacio Arganda-Carreras (https://imagej.net/AnalyzeSkeleton).

Parameters

  • Input objects
    • Type: InputObjectsP

  • Add skeletons to workspace
    • Type: BooleanP
    • Default value: false

  • Output objects
    • Type: OutputObjectsP

  • Minimum branch length
    • Type: DoubleP
    • Default value: 0.0

  • Calibrated units
    • Type: BooleanP
    • Default value: false

  • Enable multithreading
    • Type: BooleanP
    • Default value: true
