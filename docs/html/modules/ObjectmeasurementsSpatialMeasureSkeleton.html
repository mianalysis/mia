Measure skeleton

Description

Uses the AnalyzeSkeleton plugin by Ignacio Arganda-Carreras.

The optional, output skeleton object acts solely as a linking object for the edge, junction and loop objects. It doesn't itself hold any coordinate data.

Parameters

  • Input objects (default = "")

  • Add skeletons to workspace (default = "false")

  • Output skeleton objects (default = "")

  • Output edge objects (default = "")

  • Output junction objects (default = "")

  • Export loop objects (default = "true")

  • Output loop objects (default = "")

  • Minimum branch length (default = "0.0")

  • Calibrated units (default = "false") When selected, spatial values are assumed to be specified in calibrated units (as defined by the "Input control" parameter "Spatial units"). Otherwise, pixel units are assumed.

  • Enable multithreading (default = "true")
