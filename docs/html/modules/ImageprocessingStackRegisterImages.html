Register images

Description

Uses SIFT image registration toolbox

Parameters

  • Input image
    • Type: InputImageP

  • Apply to input image
    • Type: BooleanP
    • Default value: true

  • Output image
    • Type: OutputImageP

  • Transformation mode
    • Type: ChoiceP
    • Default value: Rigid
    • Choices:
      • Affine
      • Rigid
      • Similarity
      • Translation

  • Alignment mode
    • Type: ChoiceP
    • Default value: Automatic (feature extraction)
    • Choices:
      • Automatic (feature extraction)
      • Manual (landmarks)

  • Fill mode
    • Type: ChoiceP
    • Default value: Black
    • Choices:
      • Black
      • White

  • Enable multithreading
    • Type: BooleanP
    • Default value: true

  • Relative mode
    • Type: ChoiceP
    • Default value: First frame
    • Choices:
      • First frame
      • Previous frame
      • Specific image

  • Rolling correction
    • Type: ChoiceP
    • Default value: None
    • Choices:
      • None
      • Every nth frame

  • Correction interval
    • Type: IntegerP
    • Default value: 1

  • Reference image
    • Type: InputImageP

  • Calculation source
    • Type: ChoiceP
    • Default value: Internal
    • Choices:
      • Internal
      • External

  • External source
    • Type: InputImageP

  • Calculation channel
    • Type: IntegerP
    • Default value: 1

  • Initial Gaussian blur (px)
    • Type: DoubleP
    • Default value: 1.6

  • Steps per scale
    • Type: IntegerP
    • Default value: 3

  • Minimum image size (px)
    • Type: IntegerP
    • Default value: 64

  • Maximum image size (px)
    • Type: IntegerP
    • Default value: 1024

  • Feature descriptor size
    • Type: IntegerP
    • Default value: 4

  • Feature descriptor orientation bins
    • Type: IntegerP
    • Default value: 8

  • Closest/next closest ratio
    • Type: DoubleP
    • Default value: 0.92

  • Maximal alignment error (px)
    • Type: DoubleP
    • Default value: 25.0

  • Inlier ratio
    • Type: DoubleP
    • Default value: 0.05
