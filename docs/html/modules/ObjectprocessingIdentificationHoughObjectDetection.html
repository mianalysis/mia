Description

Parameters

  • Input image

  • Output objects

  • Minimum radius (px)

  • Maximum radius (px)
    • Type: IntegerP
    • Default value: 20

    • Type: DoubleP
    • Default value: 0.5

  • Detection threshold
    • Type: DoubleP
    • Default value: 1.0

  • Exclusion radius (px)
    • Type: IntegerP
    • Default value: 10

  • Show transform image
    • Type: BooleanP
    • Default value: true

  • Show detection image
    • Type: BooleanP
    • Default value: true

  • Show detection score
    • Type: BooleanP
    • Default value: false

    • Type: IntegerP
