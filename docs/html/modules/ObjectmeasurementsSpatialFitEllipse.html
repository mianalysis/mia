Parameters

  • Input objects
    • Type: InputObjectsP

    • Type: InputImageP

  • Object output mode
    • Type: ChoiceP
    • Default value: Do not store
    • Choices:

  • Output objects
    • Type: OutputObjectsP

    • Type: ChoiceP
    • Default value: Fit to surface

    • Type: DoubleP
