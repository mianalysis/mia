Watershed transform

Description

Uses MorphoLibJ implementation of watershed transform.

Parameters

  • Input image
    • Type: InputImageP
  • Apply to input image
    • Type: BooleanP
    • Default value: true
  • Output image
    • Type: OutputImageP
  • Use markers
    • Type: BooleanP
    • Default value: false
  • Input marker image
    • Type: InputImageP
  • Intensity mode
    • Type: ChoiceP
    • Default value: Distance
    • Choices:
      • Distance
      • Input image intensity
  • Intensity image
    • Type: InputImageP
  • Dynamic
    • Type: IntegerP
    • Default value: 1
  • Connectivity
    • Type: ChoiceP
    • Default value: 26
    • Choices:
      • 6
      • 26
  • Match Z to XY
    • Type: BooleanP
    • Default value: true
  • Enable multithreading
    • Type: BooleanP
    • Default value: true
