Load image

Description

Load image into MIA workspace. This module can be configured to import images from a variety of locations (selected using the "Import mode" control).

Parameters

  • Core image loading controls
    • Type: GUISeparatorP
  • Output mode
    • Type: ChoiceP
    • Default value: Image
    • Choices:
      • Image
      • Objects
  • Output image
    • Type: OutputImageP
  • Output objects
    • Type: OutputObjectsP
  • Import mode
    • Type: ChoiceP
    • Default value: Current file
    • Choices:
      • Current file
      • From ImageJ
      • Image sequence
      • Matching format
      • Specific file
  • Use ImageJ reader
    • Type: BooleanP
    • Default value: false
  • Number of zeroes
    • Type: IntegerP
    • Default value: 4
  • Starting index
    • Type: IntegerP
    • Default value: 0
  • Frame interval
    • Type: IntegerP
    • Default value: 1
  • Limit frames
    • Type: BooleanP
    • Default value: false
  • Final index
    • Type: IntegerP
    • Default value: 1
  • Name format
    • Type: ChoiceP
    • Default value: Huygens
    • Choices:
      • Huygens
      • Incucyte short filename
      • Yokogowa
      • Input filename with prefix
      • Input filename with suffix
  • Comment
    • Type: StringP
  • Prefix
    • Type: StringP
  • Suffix
    • Type: StringP
  • Extension
    • Type: StringP
  • Include series number
    • Type: BooleanP
    • Default value: true
  • File path
    • Type: FilePathP
  • Dimension ranges and cropping
    • Type: GUISeparatorP
  • Channels
    • Type: StringP
    • Default value: 1-end
  • Slices
    • Type: StringP
    • Default value: 1-end
  • Frames
    • Type: StringP
    • Default value: 1-end
  • Channel
    • Type: IntegerP
    • Default value: 1
  • Load 3D stacks as
    • Type: ChoiceP
    • Default value: Z-Stack
    • Choices:
      • Timeseries
      • Z-Stack
  • Crop mode
    • Type: ChoiceP
    • Default value: None
    • Choices:
      • None
      • Fixed
      • From reference
  • Reference image
    • Type: InputImageP
  • Left coordinate
    • Type: IntegerP
    • Default value: 0
  • Top coordinate
    • Type: IntegerP
    • Default value: 0
  • Width
    • Type: IntegerP
    • Default value: 512
  • Height
    • Type: IntegerP
    • Default value: 512
  • Spatial and intensity calibration
    • Type: GUISeparatorP
  • Set manual spatial calibration
    • Type: BooleanP
    • Default value: false
  • XY calibration (dist/px)
    • Type: DoubleP
    • Default value: 1.0
  • Z calibration (dist/px)
    • Type: DoubleP
    • Default value: 1.0
  • Force bit depth
    • Type: BooleanP
    • Default value: false
  • Output bit depth
    • Type: ChoiceP
    • Default value: 8
    • Choices:
      • 8
      • 16
      • 32
  • Minimum input intensity
    • Type: DoubleP
    • Default value: 0.0
  • Maximum input intensity
    • Type: DoubleP
    • Default value: 1.0
