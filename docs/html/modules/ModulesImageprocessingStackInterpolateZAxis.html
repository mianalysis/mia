Interpolate Z axis

Description

Interpolates Z-axis of image to match XY spatial calibration

Parameters

  • Input image (default = "") Input image to which the Z-axis interpolation will be applied.

  • Output image (default = "") Output image with Z-axis interpolation applied. This image will be stored in the workspace and be accessible using this name.
