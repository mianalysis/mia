Calculate statistics for partners

Description

Calculates statistics for a measurement associated with all partner objects of an input object. The calculated statistics are stored as new measurements, associated with the relevant input object. For example, calculating the summed volume of all partner objects (from a specified collection) of each input object.

Parameters

  • Input objects (default = "") Input object collection from the workspace for which statistics of partner object measurements will be calculated. This object collection is a partner to those selected by the "Partner objects" parameter. Statistics for one measurement associated with all partners of each input object will be calculated and added to this object as a new measurement.

  • Partner objects (default = "") Input object collection from the workspace, where these objects are partners of the collection selected by the "Input objects" parameter.)

  • Measurement (default = "") Measurement associated with the partner objects for which statistics will be calculated. Statistics will be calculated for all partners of an input object.

  • Calculate mean (default = "true") When selected, the mean value of the measurements will be calculated and added to the relevant input object.

  • Calculate standard deviation (default = "true") When selected, the standard deviation of the measurements will be calculated and added to the relevant input object.

  • Calculate minimum (default = "true") When selected, the minimum value of the measurements will be calculated and added to the relevant input object.

  • Calculate maximum (default = "true") When selected, the maximum value of the measurements will be calculated and added to the relevant input object.

  • Calculate sum (default = "true") When selected, the sum of the measurements will be calculated and added to the relevant input object.
