Threshold image

Description

Parameters

  • Input image
    • Type: InputImageP

  • Apply to input image
    • Type: BooleanP
    • Default value: true

  • Output image
    • Type: OutputImageP

  • Threshold type
    • Type: ChoiceP
    • Default value: Global
    • Choices:
      • Global
      • Local
      • Manual

  • Global threshold algorithm
    • Type: ChoiceP
    • Default value: Huang
    • Choices:
      • Huang
      • Intermodes
      • IsoData
      • Li
      • MaxEntropy
      • Mean
      • MinError
      • Minimum
      • Moments
      • Otsu
      • Percentile
      • RenyiEntropy
      • Shanbhag
      • Triangle
      • Yen

  • Local threshold algorithm
    • Type: ChoiceP
    • Default value: Phansalkar (3D)
    • Choices:
      • Bernsen (3D)
      • Contrast (3D)
      • Mean (3D)
      • Median (3D)
      • Phansalkar (3D)
      • Phansalkar (slice-by-slice)

  • Threshold multiplier
    • Type: DoubleP
    • Default value: 1.0

  • Use lower threshold limit
    • Type: BooleanP
    • Default value: false

  • Lower threshold limit
    • Type: DoubleP
    • Default value: 0.0

  • Local radius
    • Type: DoubleP
    • Default value: 1.0

  • Spatial units
    • Type: ChoiceP
    • Default value: Pixel
    • Choices:
      • Calibrated
      • Pixel

  • Threshold value
    • Type: IntegerP
    • Default value: 1

  • Use full Z-range ("Global Z")
    • Type: BooleanP
    • Default value: false

  • Black objects/white background
    • Type: BooleanP
    • Default value: true

  • Store threshold as measurement
    • Type: BooleanP
    • Default value: false
