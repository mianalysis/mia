Create distance map

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • Input image
    • Type: InputImageP

  • Output image
    • Type: OutputImageP

  • Reference mode
    • Type: ChoiceP
    • Default value: Distance from object centroid
    • Choices:
      • Distance from object centroid
      • Distance from object edge

  • Invert map within objects
    • Type: BooleanP
    • Default value: true

  • Masking mode
    • Type: ChoiceP
    • Default value: Inside and outside
    • Choices:
      • Inside and outside
      • Inside only
      • Outside only

  • Normalise map per object
    • Type: BooleanP
    • Default value: false

  • Spatial units
    • Type: ChoiceP
    • Default value: Pixel
    • Choices:
      • Calibrated
      • Pixel
