Hough-based detection

Description

Parameters

  • Input image (default = "")

  • Output objects (default = "")

  • Output transform image (default = "false")

  • Output image (default = "")

  • Minimum radius (px) (default = "10")

  • Maximum radius (px) (default = "20")

  • Sampling rate (default = "1")

  • Detection threshold (default = "1.0")

  • Exclusion radius (px) (default = "10")

  • Enable multithreading (default = "true")

  • Output radius resize (px) (default = "0") Radius of output objects will be adjusted by this value. For example, a detected circle of radius 5 with a "radius resize" of 2 will have an output of 7. Similarly, setting "radius resize" to -3 would produce a circle of radius 2.

  • Show transform image (default = "true")

  • Show detection image (default = "true")

  • Show detection score (default = "false")

  • Label size (default = "12")
