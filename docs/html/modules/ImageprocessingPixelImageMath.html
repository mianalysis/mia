Image math

Description

Applies a mathematical operation to all pixels of the input image stack. Operations that can be performed are: Absolute, Add, Divide, Multiply, Square, Squareroot, Subtract

Parameters

  • Input image (default = "")

  • Apply to input image (default = "true")

  • Output image (default = "")

  • Calculation (default = "Add")

  • Value source (default = "Fixed value")

  • Measurement (default = "")

  • Value (default = "1.0")
