Relate objects

Description

Parameters

  • Object input
    • Type: ParamSeparatorP
  • Parent (larger) objects
    • Type: InputObjectsP
  • Child (smaller) objects
    • Type: InputObjectsP
  • Relation controls
    • Type: ParamSeparatorP
  • Method to relate objects
    • Type: ChoiceP
    • Default value: Matching IDs
    • Choices:
      • Matching IDs
      • Proximity
      • Proximity to children
      • Spatial overlap
  • Reference point
    • Type: ChoiceP
    • Default value: Centroid
    • Choices:
      • Centroid
      • Surface
      • Child centroid to parent surface
  • Child objects to test against
    • Type: ChildObjectsP
  • Limit linking by distance
    • Type: BooleanP
    • Default value: false
  • Maximum linking distance (px)
    • Type: DoubleP
    • Default value: 1.0
  • Inside/outside mode
    • Type: ChoiceP
    • Default value: Inside and outside
    • Choices:
      • Inside and outside
      • Inside only (set outside to zero)
      • Outside only (set inside to zero)
  • Minimum percentage overlap
    • Type: DoubleP
    • Default value: 0.0
  • Require centroid overlap
    • Type: BooleanP
    • Default value: true
  • Only link objects in same frame
    • Type: BooleanP
    • Default value: true
  • Object output
    • Type: ParamSeparatorP
  • Merge related objects
    • Type: BooleanP
    • Default value: false
  • Output overlapping objects
    • Type: OutputObjectsP
