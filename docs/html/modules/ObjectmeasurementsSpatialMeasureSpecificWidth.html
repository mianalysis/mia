Measure specific width

Description

Measures the width of the object along the axis passing through the centroid and specified point.

Parameters

  • Input objects (default = "")

  • Reference mode 1 (default = "Object centroid")

  • Reference image 1 (default = "")

  • X-pos. image meas. 1 (px) (default = "")

  • Y-pos. image meas. 1 (px) (default = "")

  • Z-pos. image meas. 1 (slice) (default = "")

  • X-pos. object meas. 1 (px) (default = "")

  • Y-pos. object meas. 1 (px) (default = "")

  • Z-pos. object meas. 1 (slice) (default = "")

  • Reference mode 2 (default = "Object centroid")

  • Reference image 2 (default = "")

  • X-pos. image meas. 2 (px) (default = "")

  • Y-pos. image meas. 2 (px) (default = "")

  • Z-pos. image meas. 2 (slice) (default = "")

  • X-pos. object meas. 2 (px) (default = "")

  • Y-pos. object meas. 2 (px) (default = "")

  • Z-pos. object meas. 2 (slice) (default = "")

  • Measurement prefix (default = "")
