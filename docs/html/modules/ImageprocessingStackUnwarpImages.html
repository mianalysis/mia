Parameters

    • Type: OutputImageP

  • Relative mode
    • Type: ChoiceP
    • Default value: First frame
    • Choices:

    • Type: ChoiceP
    • Default value: None
    • Choices:

      • Every nth frame

    • Type: IntegerP

    • Type: InputImageP

    • Type: ChoiceP

    • Type: InputImageP

  • Calculation channel
    • Type: IntegerP

      • Accurate
      • Mono

    • Type: IntegerP
    • Default value: 0

  • Initial deformation mode
    • Type: ChoiceP
    • Default value: Very Coarse
    • Choices:
      • Very Coarse
      • Coarse
      • Fine
      • Very Fine

  • Final deformation mode
    • Type: ChoiceP
    • Default value: Fine
    • Choices:

    • Type: DoubleP
    • Default value: 0.0

  • Curl weight
    • Type: DoubleP

    • Type: DoubleP
    • Default value: 0.0

    • Type: DoubleP
    • Default value: 1.0

    • Type: DoubleP
