Fix skeleton breaks

Description

Parameters

  • Input image (default = "") Image to apply break correction to.

  • Apply to input image (default = "true") Select if the correction should be applied directly to the input image, or if it should be applied to a duplicate, then stored as a different image in the workspace.

  • Output image (default = "") Name of the output image created during processing. This image will be added to the workspace.

  • Number of end pixels to fit (default = "5") Number of pixels at the end of a branch to be used for determination of branch orientation.

  • Maximum linking distance (default = "10.0") Maximum break distance that can be bridged. Specified in pixels unless "Calibrated units" is enabled.

  • Calibrated units (default = "false") Select whether "Maximum linking distance" should be specified in pixel (false) or calibrated (true) units.

  • Maximum linking angle (degrees) (default = "45.0") Maximum angular deviation of linking region relative to orientation of existing branch end. Specified in degrees.

  • Only link ends (default = "false") Only remove breaks between pixels at branch ends. When disabled, an end can link into the middle of another branch.

  • Angle weight (default = "1.0")

  • Distance weight (default = "1.0")

  • End weight (default = "20.0")
