Image calculator

Description

Parameters

  • Image input/output
    • Type: GUISeparatorP
  • Input image 1
    • Type: InputImageP
  • Input image 2
    • Type: InputImageP
  • Overwrite mode
    • Type: ChoiceP
    • Default value: Create new image
    • Choices:
      • Create new image
      • Overwrite image 1
      • Overwrite image 2
  • Output image
    • Type: OutputImageP
  • Output 32-bit image
    • Type: BooleanP
    • Default value: false
  • Image calculation
    • Type: GUISeparatorP
  • Calculation method
    • Type: ChoiceP
    • Default value: Add image 1 and image 2
    • Choices:
      • Add image 1 and image 2
      • Difference of image 1 and image 2
      • Divide image 1 by image 2
      • Multiply image 1 and image 2
      • Subtract image 2 from image 1
  • Set NaN values to zero
    • Type: BooleanP
    • Default value: false
