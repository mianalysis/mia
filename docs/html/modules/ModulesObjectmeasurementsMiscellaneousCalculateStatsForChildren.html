Calculate statistics for children

Description

Calculates statistics for a measurement associated with all child objects of parent object. The calculated statistics are stored as new measurements, associated with the relevant parent object. For example, calculating the summed volume of all child objects (from a specified collection) of each parent object.

Parameters

  • Parent objects (default = "") Input object collection from the workspace for which statistics of child object measurements will be calculated. This object collection is a parent to those selected by the "Child objects" parameter. Statistics for one measurement associated with all children of each input parent object will be calculated and added to this object as a new measurement.

  • Child objects (default = "") Input object collection from the workspace, where these objects are children of the collection selected by the "Parent objects" parameter.)

  • Measurement (default = "") Measurement associated with the child objects for which statistics will be calculated. Statistics will be calculated for all children of a parent object.

  • Calculate mean (default = "true") When selected, the mean value of the measurements will be calculated and added to the relevant parent object.

  • Calculate standard deviation (default = "true") When selected, the standard deviation of the measurements will be calculated and added to the relevant parent object.

  • Calculate minimum (default = "true") When selected, the minimum value of the measurements will be calculated and added to the relevant parent object.

  • Calculate maximum (default = "true") When selected, the maximum value of the measurements will be calculated and added to the relevant parent object.

  • Calculate sum (default = "true") When selected, the sum of the measurements will be calculated and added to the relevant parent object.
