Get object surface

Description

Create surface objects for each input object. Surface coordinates are those with at least one non-object neighbouring pixel (using 26-way connectivity). Surfaces are stored as children of the input object.

Parameters

  • Input objects
    • Description: Input objects to extract surface from.
    • Type: InputObjectsP

  • Output objects
    • Description: Output surface objects to be stored in the workspace.
    • Type: OutputObjectsP
