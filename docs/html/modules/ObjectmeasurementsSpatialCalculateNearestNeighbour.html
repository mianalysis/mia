Calculate nearest neighbour

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • RelationshipRef mode
    • Type: ChoiceP
    • Default value: Within same object set
    • Choices:
      • Within same object set
      • Different object set

  • Neighbour objects
    • Type: InputObjectsP

  • Only calculate for objects in same parent
    • Type: BooleanP
    • Default value: false

  • Parent objects
    • Type: ParentObjectsP

  • Limit linking distance
    • Type: BooleanP
    • Default value: false

  • Maximum linking distance
    • Type: DoubleP
    • Default value: 100.0

  • Calibrated distance
    • Type: BooleanP
    • Default value: false
