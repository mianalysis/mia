Load objects

Description

Parameters

  • Output objects
    • Type: OutputObjectsP

  • Coordinate source
    • Type: ChoiceP
    • Default value: Current file
    • Choices:
      • Current file
      • Matching format
      • Specific file

  • Name format
    • Type: ChoiceP
    • Default value: Generic (from metadata)
    • Choices:
      • Generic (from metadata)
      • Input filename with prefix
      • Input filename with suffix

  • Generic format
    • Type: StringP

  • Available metadata fields
    • Type: TextAreaP

  • Prefix
    • Type: StringP

  • Suffix
    • Type: StringP

  • Extension
    • Type: StringP

  • Include series number
    • Type: BooleanP
    • Default value: true

  • Input file
    • Type: FilePathP

  • ID-column index
    • Type: IntegerP
    • Default value: 0

  • X-column index
    • Type: IntegerP
    • Default value: 1

  • Y-column index
    • Type: IntegerP
    • Default value: 2

  • Z-column index
    • Type: IntegerP
    • Default value: 3

  • Timepoint-column index
    • Type: IntegerP
    • Default value: 4

  • Limits source
    • Type: ChoiceP
    • Default value: From image
    • Choices:
      • From image
      • Manual
      • Maximum coordinate

  • Limits reference image
    • Type: InputImageP

  • Width
    • Type: IntegerP
    • Default value: 512

  • Height
    • Type: IntegerP
    • Default value: 512

  • Number of slices
    • Type: IntegerP
    • Default value: 1

  • Number of timepoints
    • Type: IntegerP
    • Default value: 1

  • Calibration source
    • Type: ChoiceP
    • Default value: From image
    • Choices:
      • From image
      • Manual

  • Calibration reference image
    • Type: InputImageP

  • XY cal (dist/px)
    • Description: Distance per pixel in the XY plane. Units for this are specified in the main "Input control" module.
    • Type: DoubleP
    • Default value: 1.0

  • Z cal (dist/px)
    • Description: Distance per slice (Z-axis). Units for this are specified in the main "Input control" module.
    • Type: DoubleP
    • Default value: 1.0

  • Create parent objects
    • Type: BooleanP
    • Default value: false

  • Parent objects type
    • Type: ChoiceP
    • Default value: Normal
    • Choices:
      • Cluster
      • Normal
      • Track

  • Output parent clusters name
    • Type: OutputClusterObjectsP

  • Output parent objects name
    • Type: OutputObjectsP

  • Output parent tracks name
    • Type: OutputTrackObjectP

  • Parent object ID index
    • Type: IntegerP
    • Default value: 5
