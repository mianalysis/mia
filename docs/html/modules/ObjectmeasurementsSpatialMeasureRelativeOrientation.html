Measure relative orientation

Description

Currently only works for X-Y plane measurements

Parameters

  • Input objects
    • Type: InputObjectsP

  • Orientation mode
    • Type: ChoiceP
    • Default value: Orientation in X-Y plane
    • Choices:
      • Orientation in X-Y plane

  • Orientation in X/Y measurement
    • Type: ObjectMeasurementP

  • Orientation in XY/Z measurement
    • Type: ObjectMeasurementP

  • Measurement range
    • Type: ChoiceP
    • Default value: 0-90 degrees
    • Choices:
      • 0-90 degrees
      • 0-180 degrees

  • Reference mode
    • Type: ChoiceP
    • Default value: Object centroid to image centre
    • Choices:
      • Object centroid to image centre
      • Object centroid to target centroid
      • Object centroid to target surface

  • Reference objects
    • Type: InputObjectsP

  • Object choice mode
    • Type: ChoiceP
    • Default value: Largest object
    • Choices:
      • Largest object
      • Smallest object

  • Reference must be in same frame
    • Type: BooleanP
    • Default value: true
