Output control

Description

Controls data export for each analysis run. Generally, data export doesn't occur until all queued jobs have completed; however, it is possible to export the current version of

Parameters

  • Run macro
    • Type: BooleanP
    • Default value: false

  • Add variable
    • Type: ParameterGroup

  • Macro mode
    • Type: ChoiceP
    • Default value: Macro text
    • Choices:
      • Macro file
      • Macro text

  • Macro text
    • Type: TextAreaP
    • Default value: run("Enable MIA Extensions"); // Get a list of Workspace IDs with Ext.MIA_GetListOfWorkspaceIDs() and set active workspace with Ext.MIA_SetActiveWorkspace(ID).

  • Macro file
    • Type: FilePathP

  • Export mode
    • Type: ChoiceP
    • Default value: All together
    • Choices:
      • All together
      • Group by metadata
      • Individual files
      • None

  • Individual save location
    • Type: ChoiceP
    • Default value: Save with input file
    • Choices:
      • Mirrored directory
      • Save with input file
      • Specific location

  • Group save location
    • Type: ChoiceP
    • Default value: Save with input file
    • Choices:
      • Save with input file
      • Specific location

  • File path
    • Type: FolderPathP

  • Mirrored directory root
    • Type: FolderPathP

  • Save name mode
    • Type: ChoiceP
    • Default value: Match input file/folder name
    • Choices:
      • Match input file/folder name
      • Specific name

  • File name
    • Type: StringP

  • Metadata item for grouping
    • Type: MetadataItemP

  • Continuous data export
    • Type: BooleanP
    • Default value: false

  • Save every n files
    • Type: IntegerP
    • Default value: 10

  • Append date/time mode
    • Type: ChoiceP
    • Default value: Never
    • Choices:
      • Always
      • If file exists
      • Never

  • Export summary
    • Type: BooleanP
    • Default value: true

  • Summary mode
    • Type: ChoiceP
    • Default value: Per input file
    • Choices:
      • Per input file
      • Per timepoint per input file
      • Group by metadata

  • Metadata item for summary
    • Type: MetadataItemP

  • Show object counts
    • Type: BooleanP
    • Default value: true

  • Export individual objects
    • Type: BooleanP
    • Default value: true
