Resolve co-occurrence

Description

Identifies overlapping objects and moves them to a new object collection Objects are linked on a one-to-one basis.

Parameters

  • Input objects 1
    • Type: InputObjectsP

  • Input objects 2
    • Type: InputObjectsP

  • Output objects name
    • Type: OutputObjectsP

  • Overlap mode
    • Type: ChoiceP
    • Default value: Spatial overlap
    • Choices:
      • Centroid separation
      • Spatial overlap

  • Maximum separation
    • Type: DoubleP
    • Default value: 1.0

  • Calibrated units
    • Type: BooleanP
    • Default value: false

  • Minimum overlap of object 1 (%)
    • Type: DoubleP
    • Default value: 50.0

  • Minimum overlap of object 2 (%)
    • Type: DoubleP
    • Default value: 50.0
