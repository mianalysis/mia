Ridge detection

Description

Uses the RidgeDetection Fiji plugin by Thorsten Wagner, which implements Carsten Steger's paper "An Unbiased Detector of Curvilinear Structures" INCOMPLETE

Parameters

  • Input image
    • Type: InputImageP
  • Output objects
    • Type: OutputObjectsP
  • Lower threshold
    • Type: DoubleP
    • Default value: 0.5
  • Upper threshold
    • Type: DoubleP
    • Default value: 0.85
  • Sigma
    • Type: DoubleP
    • Default value: 3.0
  • Calibrated units
    • Type: BooleanP
    • Default value: false
  • Estimate width
    • Type: BooleanP
    • Default value: false
  • Contour contrast
    • Type: ChoiceP
    • Default value: Dark line
    • Choices:
      • Dark line
      • Light line
  • Minimum length
    • Type: DoubleP
    • Default value: 0.0
  • Maximum length
    • Type: DoubleP
    • Default value: 0.0
  • Link contours
    • Type: BooleanP
    • Default value: false
