Normalise intensity

Description

Sets the intensity to maximise the dynamic range of the image. "Clipping fraction" is the fraction of pixels at either end of the range that gets clipped.The "Per object" region mode will normalise all pixels within each object.

Parameters

  • Image input/output
    • Type: ParamSeparatorP
  • Input image
    • Type: InputImageP
  • Apply to input image
    • Type: BooleanP
    • Default value: true
  • Output image
    • Type: OutputImageP
  • Output controls
    • Type: ParamSeparatorP
  • Region mode
    • Type: ChoiceP
    • Default value: Entire image
    • Choices:
      • Entire image
      • Per object
  • Input objects
    • Type: InputObjectsP
  • Intensity normalisation
    • Type: ParamSeparatorP
  • Calculation mode
    • Type: ChoiceP
    • Default value: Fast
    • Choices:
      • Fast
      • Manual
      • Precise
  • Clipping fraction
    • Type: DoubleP
    • Default value: 0.0
  • Minimum range value
    • Type: DoubleP
    • Default value: 0.0
  • Maximum range value
    • Type: DoubleP
    • Default value: 255.0
